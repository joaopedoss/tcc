Guia de Como Utilizar

Como Utilizar o Terrário Automatizado com Arduino Uno Ligar o Terrário: Certifique-se de que o terrário esteja conectado à energia. O Arduino Uno deve estar ligado. Monitorar o Display (opcional): Se você tiver um display LCD, ele mostrará informações como temperatura, umidade e status das funções. Caso contrário, você pode pular para o próximo passo. Controle das Funções: O terrário possui três funções principais: Iluminação Automática: Durante o dia, a luz será ativada automaticamente. À noite, a luz será desligada. Isso ajuda a simular o ciclo natural de luz para as plantas. Aquecimento Automático: O relé de calor liga a manta térmica para manter a temperatura adequada. O Arduino Uno monitora a temperatura e ativa o relé conforme necessário. Irrigação Automática: O relé de chuva liga a bomba de água para regar as plantas. O Arduino Uno pode ser programado para regar as plantas em intervalos específicos (por exemplo, a cada 2 horas). Controle via Aplicativo: Desenvolva um aplicativo para controlar o terrário. O aplicativo deve se comunicar com o Arduino Uno via Bluetooth, Wi-Fi ou outro meio. Implemente botões para ligar/desligar as funções (luz, calor, chuva). Instruções para os Usuários: Abra o aplicativo. Conecte-se ao terrário via Bluetooth/Wi-Fi. Use os botões no aplicativo para ativar/desativar as funções conforme necessário. Manutenção Regular: Verifique se todos os componentes estão funcionando corretamente. Monitore os níveis de umidade e temperatura. Substitua a água da bomba de chuva conforme necessário.
